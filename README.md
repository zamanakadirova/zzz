## Қаржы бақылау жүйесі (Finance Tracker)

# Жоба сипаттамасы

Finance Tracker – кіріс және шығыс қаржы операцияларын басқаруға арналған React негізіндегі веб-қосымша. Бұл жүйе қаржы ағынын бақылауға, графиктер арқылы көруге және деректерді экспорттауға мүмкіндік береді.

# Қолданылатын технологиялар

React – қолданбалы интерфейс құру үшін

Recharts – қаржы деректерін визуалдау үшін

Axios – API-дан деректер алу үшін

FileSaver.js – деректерді CSV және JSON форматында жүктеу үшін

jsPDF & jspdf-autotable – деректерді PDF ретінде экспорттау үшін
